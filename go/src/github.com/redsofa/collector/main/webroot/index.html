Coloring My SoundTouch Events

What is this visualization ? It's a representation of the live WebSocket notifications that are sent from my Bose SoundTouch.
As we interact with our SoundTouch at home, it's generating notifications. These notifications are captured and displayed here.
Hover over the dots in the The SoundTouch Events visualization to see what kinds of events these dots correspond to.
The Rolling Events Log keeps track of the last 10 events. What kinds of events do I care about ?

  • Volume Changes
  • Preset Updates
  • Socket Connections
  • SoundTouch Heartbeats
  • Recently Played Updates
